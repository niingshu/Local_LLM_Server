# HomeLLM
Self-hosted LLM inference server running Qwen2.5-1.5B, accessible from any device via private Tailscale VPN Tunnel. Containerize with Docker, orchestrated with Kubernetes (k3s) on AWS EC2, and automatically deployed via GitHub Actions CI/CD

## Architecture 


## Tech Stack 
- **LLM:** Qwen2.5-1.5B via Ollama 
- **Frontend:** Open WebUI
- **Containerization:** Docker
- **Orchestration:** Kubernetes (k3s)
- **Cloud:** AWS EC2 (t3.small) with Elastic IP
- **Networking:** Tailscale VPN
- **CI/CD:** GitHub Actions
